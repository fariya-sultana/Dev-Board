Question-1: What are the different ways to select an element in the DOM?

There are several ways to select an element in the DOM:getElementById(): Selects by ID,getElementsByClassName(): Selects by class name,getElementsByTagName(): Selects by tag name,querySelector(): Selects the first matching element using CSS selectors,querySelectorAll(): Selects all matching elements.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

InnerHTML: Gets or sets the HTML content, including tags, inside an element,innerText: Gets or sets only the visible text content, ignoring hidden elements,textContent: Gets or sets all text content, including hidden text, without HTML tags.

Question-3: What is event delegation in the DOM?

Event delegation is a technique in the DOM where a parent element handles events for its child elements. Instead of adding event listeners to each child, you add one to the parent. It works using event bubbling, making code efficient.

Question-4: What is event bubbling in the DOM?

Event bubbling means when you click an element, the event first happens on that element and then moves up to its parent and other ancestors unless stopped.

Question-5: How do you create, add, and remove elements using JavaScript?

To create an element, use document.createElement(). To add it, use appendChild(). To remove an element, use removeChild(). This helps in dynamically updating web pages.
