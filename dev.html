Question-1: What are the different ways to select an element in the DOM?

JavaScript provides multiple ways to select DOM elements:getElementById("id") – Selects by ID,getElementsByClassName("class") – Selects by class,querySelector("selector") – Selects the first match,querySelectorAll("selector") – Selects all matches,getElementsByTagName("tag") – Selects by tag.
